1.窗口是flink用来将无限数据流切割成有限的数据块，其实窗口是一个一个的桶，每个数据都会落入到对应的桶中，当到达窗口结束时间，就会触发窗口的计算。
2.窗口的分类：
    2.1  按照驱动类型区分，分为时间窗口和计数窗口
    2.2  按照窗口分配器来区分：分为滚动窗口，滑动窗口，会话窗口，全局窗口
3.窗口分配器：
    时间窗口：
        事件时间:滚动事件时间窗口，滑动事件时间窗口，会话事件时间窗口
        处理时间:滚动处理时间窗口，滑动处理时间窗口，会话处理时间窗口
    计数窗口：
        滚动计数窗口，滑动计数窗口
窗口函数：
    1.增量聚合函数（ReduceFunction / AggregateFunction）
    2.全窗口函数
